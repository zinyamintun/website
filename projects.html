VirtuSlide: Haptic Shuffleboard

I developed a haptic-enabled virtual shuffleboard game in Unity, integrating real-time physics and force feedback with a custom 2-DOF pantograph interface via serial communication of user input (position, velocity, button state). To deliver realistic tactile sensations, I implemented physics-based rendering of mass inertia, Coulomb and viscous friction, and boundary collisions using a virtual coupling and spring-damper model.

Touch Sensing Display

I developed a C++ algorithm for a mutual-capacitance touch sensing system using a Raspberry Pi and AD/DA hat. The system uses pseudorandom binary sequence (PRBS) signal generation, real-time signal correlation for capacitance mapping, and centroid detection to estimate touch location with high accuracy.

Fast Robot

I assembled a mobile robot by soldering an Artemis board, Time of Flight sensors, IMU, and motor drivers to build a robot from scratch. I developed algorithms in Python and C++ for Bluetooth communication of sensor readings, enabling PID control, Kalman filtering, mapping, localization, and path planning to perform stunts and autonomous navigation.

Grape Picking Robot

With a group of four, we performed grape picking tasks using ROS (Robot Operating System) on the WidowX 250S robot. I contributed to the implementation of RRT (Rapidly-exploring Random Tree) planning and PID (Proportional-Integral-Derivative) control algorithms to successfully pick grapes in a real-time simulated environment.

Robot Perception

I leveraged PyTorch and TensorFlow to build and train basic convolutional neural networks (CNNs) for tasks such as image segmentation and pose estimation. Additionally, I implemented Truncated Signed Distance Fields (TSDF) for 3D reconstruction, integrating it with the Iterative Closest Point (ICP) algorithm to estimate object poses. This integration enabled accurate pose estimation in a simulated environment, which was then used for grasp planning in robotic tasks.

Page Turner

I collaborated with a fellow student to design a foot-pedal-operated page turner, allowing pianists to turn pages hands-free during performances. To gather design insights, we conducted interviews with three pianists, which informed the development process. The design was then modeled in Fusion 360, and the prototype was fabricated using a combination of laser cutting, 3D printing, and commercial off-the-shelf parts. Our project was recognized as the "Best Project" within our lab session based on peer voting.
